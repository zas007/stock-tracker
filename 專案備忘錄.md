## 對照分析欄位說明

### 各法人欄位（外資 / 投信 / 自營商各一組）

| 欄位 | 說明 |
|------|------|
| 累計天數 | 出現在買超前十的總天數（含不連續） |
| 連續天數 | 三法人合計淨買超 > 0 的連續天數，賣超當天歸零 |
| 買超(張) | FIFO 扣減賣超後的剩餘持倉張數 |
| 加權均價 | 價格升序 FIFO 扣減後的剩餘持倉加權均價 |
| 趨勢 | 📈遞增中 / 📉遞減中 / ➡️持平 |
| 投信標記 | 🔥連續 ≥5 天 / ⭐連續 ≥3 天 |

### 綜合欄位

| 欄位 | 說明 |
|------|------|
| 現價 | 當日收盤價 |
| 漲幅% | (現價 - 三法人綜合加權均價) ÷ 加權均價 |
| 出貨風險 | 🟢低(<10%) / 🟡中(10~15%) / 🔴高(>15%) |
| 訊號 | ✅持續買入 / ⚠️已停止買入 / 🔴今日賣超 |
| 籌碼集中度% | 三法人合計買超張數 ÷ 當日成交量 |
| 籌碼集中度評級 | 🔵高度集中(≥20%) / 🟦中度集中(≥10%) / ⬜偏低 |
| 融資餘額(張) | 當日融資餘額 |
| 融資增減(張) | 當日融資增減（正=增加） |
| 融券餘額(張) | 當日融券餘額 |
| 融資健康度 | ✅籌碼乾淨 / 🟡小幅跟進 / ⚠️散戶大量跟進 / 🔴法人不買散戶買 |

## 族群聯動說明

- `config.py` 的 `SECTOR_MAP` 定義族群成員
- 買超榜出現任一族群成員，整個族群全員當日行情列出
- **自動補入**：買超榜出現不在 `SECTOR_MAP` 的新股票時，查 TWSE 官方產業別自動歸類並寫回 `config.py`
- 名稱從 `config.py` 的 `CODE_NAME_MAP` 取得，未定義者 fallback 到買超榜名稱
