## 對照分析欄位說明

| 欄位 | 說明 |
|---|---|
| 外資累計天數 | 出現在買超前十的總天數（含不連續） |
| 外資連續天數 | 最近連續出現的天數（斷掉歸零） |
| 外資買超(張) | 累計買超張數 |
| 外資加權均價 | Σ(買超張數×當日均價) ÷ Σ(買超張數) |
| 外資趨勢 | 📈遞增中 / 📉遞減中 / ➡️持平 |
| 現價 | 當日收盤價 |
| 漲幅% | (現價 - 三法人綜合加權均價) ÷ 加權均價 |
| 出貨風險 | 🟢低(<10%) / 🟡中(10~15%) / 🔴高(>15%) |
| 訊號 | ✅持續買入 / ⚠️已停止買入 / 🔴今日賣超 |

---

## 出貨風險邏輯
- **現價 vs 三法人綜合加權均價**的漲幅
- < 10%：🟢 低風險
- 10% ~ 15%：🟡 中風險
- > 15%：🔴 高風險（法人有足夠獲利空間，出貨動機出現）
