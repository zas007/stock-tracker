## 對照分析欄位說明

### 各法人欄位（外資 / 投信 / 自營商各一組）
| 欄位 | 說明 |
|------|------|
| 累計天數 | 出現在買超前十的總天數（含不連續） |
| 連續天數 | 三法人合計淨買超 > 0 的連續天數，賣超當天歸零 |
| 買超(張) | 累計買超張數 |
| 加權均價 | Σ(買超張數 × 當日均價) ÷ Σ(買超張數) |
| 趨勢 | 📈遞增中 / 📉遞減中 / ➡️持平 |
| 投信標記 | 🔥連續 ≥5 天 / ⭐連續 ≥3 天 |

### 綜合欄位
| 欄位 | 說明 |
|------|------|
| 現價 | 當日收盤價 |
| 漲幅% | (現價 - 三法人綜合加權均價) ÷ 加權均價 |
| 出貨風險 | 🟢低(<10%) / 🟡中(10~15%) / 🔴高(>15%) |
| 訊號 | ✅持續買入 / ⚠️已停止買入 / 🔴今日賣超 |
| 籌碼集中度% | 三法人合計買超張數 ÷ 當日成交量 |
| 籌碼集中度評級 | 🔵高度集中(≥20%) / 🟦中度集中(≥10%) / ⬜偏低 |
| 融資餘額(張) | 當日融資餘額 |
| 融資增減(張) | 當日融資增減（正=增加） |
| 融券餘額(張) | 當日融券餘額 |
| 融資健康度 | ✅籌碼乾淨 / 🟡小幅跟進 / ⚠️散戶大量跟進 / 🔴法人不買散戶買 |

## 族群聯動說明
- 設定在 `config.py` 的 `SECTOR_MAP`
- 買超榜出現任一族群成員，整個族群全員當日行情列出
- 顯示欄位：代號、股票名稱、收盤價、漲跌幅%、成交量(張)、是否在買超榜
- 名稱從 `config.py` 的 `CODE_NAME_MAP` 取得

---

## 出貨風險邏輯
- **現價 vs 三法人綜合加權均價**的漲幅
- < 10%：🟢 低風險
- 10% ~ 15%：🟡 中風險
- > 15%：🔴 高風險（法人有足夠獲利空間，出貨動機出現）

---

## 連續天數計算邏輯（v9 起）
- 以「三法人合計淨買超 > 0」判斷，不只看單一法人
- 任一交易日合計淨買超 ≤ 0（含賣超）即中斷歸零
- 外資買 500、投信賣 200 → 合計 +300 → 繼續連續
- 外資買 500、投信賣 600 → 合計 -100 → 當天中斷

- 均價計算：成交金額 ÷ 成交股數（元/股）
